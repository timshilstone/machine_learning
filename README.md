# West Nile Virus Machine Learning Project

Problem Statement: 
- Chicago experiences a West Nile Virus outbreak every year. The city government is interested in understanding what causes the outbreaks and how we can control the virus as efficiently as possible.

Data Sources: 
- Weather_df is weather data for each day pulled from two different weather stations. This data includes min and max temperatures, precipitation, wind direction and more.
- Train_df includes data from mosquito traps spread around the city. This includes the species of mosquitos caught, number of mosquitos caught, if West Nile Virus was present, the latitude and longitude of the traps and more.
